DIY Rain drop sensor
==================

Cet outil émet une petite mélodie lorsque le contact électrique est effectué sur le détecteur d'eau.

- [Matériel et coût](#matériel)
- [Schéma](#schéma)


## Matériel

- Arduino nano V3.0
- Buzzer 8 ohm
- Pile 9V 
- Resistances 1x 2.2k Ohm + 1x 3M Ohm
- Interrupteur
- Câbles
- Détecteur d'eau
